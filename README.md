## 🚀 Como Usar

1. **Clone o repositório:**
```bash
git clone https://github.com/annaconceicao/annabe.git
cd annabe
```

2. **Abra o arquivo `index.html` em seu navegador:**
   - Abra com um servidor local (recomendado)
   - Ou clique diretamente no arquivo

3. **Customize o conteúdo:**
   - Edite o `index.html` com suas informações
   - Adicione seus projetos reais
   - Atualize os links de contato

## ✏️ Personalizando

### 🎨 Cores
Edite as variáveis CSS em `styles.css`:

```css
:root {
    --primary-color: #6366f1;        /* Roxo */
    --secondary-color: #ec4899;      /* Rosa */
    --accent-color: #14b8a6;         /* Verde Teal */
    --dark-bg: #0f172a;              /* Fundo escuro */
    --light-bg: #f8fafc;             /* Fundo claro */
    --text-dark: #1e293b;            /* Texto escuro */
    --text-light: #64748b;           /* Texto claro */
}
```

### 📝 Conteúdo
1. Abra `index.html`

3. Atualize os links de projetos
4. Customize as descrições

## 🌐 Hospedagem no GitHub Pages

1. Vá para **Settings** do repositório
2. Clique em **Pages** (no menu esquerdo)
3. Em "Build and deployment", selecione:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Clique em **Save**

Seu site estará disponível em: `https://annaconceicao.github.io/annabe`
